# kiss-team
KISS Team. **KISS** = Keep It Simple & Stupid

## 动机和目标 (Motivation & Objective)

关注团队建设，追求极简主义.

## 项目范围 (Scope)

此项目包含：

  - 核心理念
  - 编码风格
  - 学习路线图
  - 推荐读物

此项目**不**包含：

  - 具体项目
  - 学习教程
