# Markdown

## 基本 (Basic)

### 块元素 (block)

- 标题 (Header)
- 段落 (Paragraph)
- 区块引用 (Blockquote)
- 列表：有序，无序，嵌套 (List: Ordered, Unordered, Nested)
- 代码块 (Code)
- 图 (Image)

### 内嵌元素 (inline)

- 链接 (Link)
- 强调 (Emphasis)
- 内嵌代码 (Inline Code)

## GitHub拓展 (GitHub Flavored Markdown - GFM)

- 任务列表 (Task List)
- 表格 (Table)
- 引用提交 (Commit SHA Reference)
- 引用Issue (Issue Reference)
- 提醒某人 (User Mention)
- 自动链接 (Automatic Linking)
- 代码语法高亮 (Code Syntax Highlighting)
- 表格 (Table)
- 删除线 (Strikethrough)
- 混合HTML (Mix HTML)
